# 🐙 OctopusPro (系统可用功能与实操指南手册)

欢迎使用 **OctopusPro** 数据决策与大模型量化研判系统。本手册旨在帮助您快速掌握系统的核心可用功能、操作流程以及日常一键运维。

---

## 🌟 核心可用功能清单

### 1. 📅 赛事日程与即时状态看板
- **多日期日程切换**：左侧提供 15 天内的日历快速切换，一键拉取指定日期的所有比赛列表。
- **即时比分与状态**：比赛列表实时显示红黄牌、即时比分、半场比分、比赛进行时点（如“下半场”、“已结束”）。
- **极速增量数据同步**：右上角提供“同步最新赛事”按钮，支持秒级拉取最新比分，且合并机制保证不会擦除您已经生成的 AI 分析和概率数据。

### 2. 🔍 五大黄金数据研判面板
当您在列表中点击任意一场比赛，右侧区域将立即为您拉取并呈现该场比赛的五大决策维度：
1. **📊 预测 & SWOT 情报**：呈现独家主客队有利/不利情报分析，提取攻防核心爆点。
2. **⚔️ 对决 & 近期战绩**：展示两队近 10 场历史交锋比分和各自近期 10 场赛果。
3. **🏥 伤停 & 阵容**：列出双方最新伤停人员、缺席原因以及目前的出场状态。
4. **📈 盘口 & 走势指数**：
   - 汇聚了主流机构的**胜平负、让球、总进球（大小球）**的初始与即时赔率/盘口水位。
   - **破风控 Canvas 指数解密**：点击任意主流机构（如 `36*`、`皇*`），系统会触发破风控解密，将由于防爬而混淆成 Canvas 图表的折线走势图在网页中央进行高精度还原。同时，拉取并呈现**让球、胜平负和大小球的三合一最近 10 次变盘时间轴历史明细**。
5. **🧠 AI 预测分析**：
   - 大模型流式分析：采用打字机效果流式输出深度研判报告（攻防战术、伤停影响对冲、庄家操盘意图解析等）。
   - **概率期望对比**：若雷速端缺少同盘口数据，AI 会基于自身强大的足球历史大数据库，自动为您匹配 2-3 场相同初盘且临场相同走势的已完场经典赛事（如一球退半球高水），统计得出历史胜平负真实比例，从概率学期望上提供更具倾斜性的博弈投注建议。

### 3. 💾 智能分析本地持久化缓存
- **分析成功即缓存**：只要您对某场比赛生成了 AI 研判，结果就会自动作为 JSON 文件持久化在服务器端。
- **切换场次自动秒开**：当您再次切换回这篇赛事，或者切换到 AI 选项卡时，**前端会全自动、零延迟地将已有报告秒级渲染出来，不再需要重新点击生成，100% 节省大模型 API 额度**。
- **支持手动强制刷新**：AI 页面右上角提供“刷新本场”按钮。如果由于临场水位发生了剧烈变化，您可以点击“刷新本场”强制大模型重新获取最新水位并生成最新的研判报告。

## 🛠️ 日常实操指南

### 1. ⚙️ 配置您的 AI 大模型参数
首次使用或更换 Key 时，请点击页面右上角的 ⚙️ **AI配置** 按钮：
- **API Key**：填入您的 OpenCode Zen API 密钥。
- **API Base**：默认已为您填好高效的 `https://opencode.ai/zen/v1` 地址。
- **选择大模型**：推荐选择免费且效果惊艳的 `deepseek-v4-flash-free`。
- **全局系统分析提示词 (System Prompt)**：系统已为您**默认注入了足球操盘大师级 Prompt**。您可以在这里直接手动修改您的分析指令，**点击保存配置后，下一次大模型分析会 100% 立即采用您的新提示词生效！**

### 2. 🔄 以后本地改了代码，如何在服务器上一键热更新
我们在项目根目录下为您编写了 [deploy.sh](deploy.sh) 自动化运维脚本。当您在本地开发了新功能并 `git push` 到 GitHub 后，**只需在服务器终端的项目目录下执行以下一句命令**：
```bash
sh deploy.sh
```
> 脚本将全自动使用您配置好的 SSH 密钥拉取最新提交，并在 1 秒内重启载入新版本，实现零摩擦、无感热升级！

### 3. 📊 如何在服务器上查看后台抓取日志
如果您想实时追踪 Playwright 无头浏览器在后台的 WAF 破盾以及数据抓取细节，可以在服务器终端执行以下命令：
```bash
sudo journalctl -u leisu -f
```
*(可以通过键盘组合键 `Ctrl + C` 随时退出日志查看。)*
